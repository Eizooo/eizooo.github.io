Jan 23, 2020

New Gameplay Glyph

Q1. What's is Glyph?
1. The Glyph level point is from 0 to 100
2. Purple and above gears can be drilled before they can be embedded with Glyphs.
3. There is a Glyph type restriction of the gear. Different gears can be embedded with different Glyphs. For example, the armor and weapon could be embedded with the Fire Glyph, the boots and weapon could be embedded with the Wind Glyph etc.
4. The Glyphs have different basic attribute, for example, the Fire Glyph could add the attack and the Wind Glyph could add the attack speed.
5. A Glyph has basic attributes and random attributes. The basic attibutes are fixed and the extra attributes are random.
6. The original rank of the Glyph is 0, which can be increased by enchantment. Enchantment requires Dusts, which can be gained from decomposing the Glyphs. The rank affects the basic attributes and the cap of the extra attributes. The quality of an attribute is indicated by color.
